## Usage

- `/heroes`: homepage and page-level hero imagery.
- `/projects`: project listing and project detail hero imagery.
- `/units`: unit detail galleries and unit cards.
- `/districts`: district landing/banner imagery.
- `/floorplans`: illustrative 2D/3D floor plan placeholders.
- `/vr`: future VR placeholder imagery.
